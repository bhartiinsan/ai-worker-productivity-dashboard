## How to Add Screenshots

1. **From the conversation:**
   - Right-click each screenshot image
   - Select "Save image as..."
   - Save to this folder with the names above

2. **From your browser:**
   - Visit http://localhost:3000
   - Press F12 to open DevTools
   - Use screenshot feature or Windows + Shift + S
   - Save with recommended filenames

3. **Using Snipping Tool:**
   - Press Windows + Shift + S
   - Capture each section
   - Paste into Paint or image editor
   - Save to this folder

## Recommended Filenames

- `dashboard-overview.png` - Top section with KPIs
- `workstations-heatmap.png` - Middle section with stations
- `ai-events-stream.png` - Bottom section with events
- `full-dashboard.png` - Complete page view

## Image Specifications

- Format: PNG (recommended)
- Resolution: 1920x1080 or actual screen resolution
- Include: Full dashboard sections with live data
- Avoid: Loading states, empty data states
